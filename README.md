## 核心功能，挨个说说

### 测距离

这是最常用的功能。选中「查找尺寸」模式，鼠标滑到任意位置，两个元素之间的像素距离就实时显示在屏幕上。低对比度的元素？可以手动调节「容差模式」来适配阴影和渐变边缘。

### 测对象尺寸

在任意元素周围框选，PixelSnap 会自动磁吸到元素边界，立刻显示宽高。一次框多个对象也没问题。比自己手动拖标尺快了不止一个档次。

### 添加参考线

可以给屏幕加上水平和垂直参考线，参考线会自动吸附到元素边缘。开启「十字准线模式」还能一次性检查整个屏幕的对齐情况，强迫症友好。

### 截图导出

测量完之后可以直接截图保存，会带上测量标注。如果你装了 CleanShot X，两者深度整合，一键截图、编辑、分享，流程极顺。

### 设计软件整合

这点很关键：PixelSnap 和 Sketch、Figma、Adobe XD、Affinity Suite 高度整合，无论设计文件里缩放比例是多少，PixelSnap 显示的都是实际像素尺寸，不会出现"量出来的数据和设计稿对不上"这种尴尬。
